2. **Live syncing** 
3. **Faster speeds with Rust**  

## **Tutorial**
1. Go to **RojoTree**:
    ```bash
    cd Tools/RojoTree
    ```

2. Initialize **RojoTree**:
    ```bash
    cargo run
    ```

3. Start **Rojo**:
    ```bash
    rojo serve
    ```
