
## Getting Started

### Prerequisites

- [Rojo](https://rojo.space/) installed
- [Roblox Studio](https://www.roblox.com/create)

### Setup

1. **Clone the repository**
```bash
   git clone 
   cd Rojo-Template
```

2. **Start RojoTree** (in one terminal)
```bash
   .\RojoTree.exe
```

3. **Start Rojo** (in another terminal)
```bash
   rojo serve
```
