# RojoTree
 Using [Rojo](https://github.com/rojo-rbx/rojo) 

# How To Use
1. Go to **RojoTree**:
    ```bash
    cd Tools/RojoTree
    ```

2. Initialize **RojoTree**:
    ```bash
    cargo run
    ```

3. Start **Rojo**:
    ```bash
    rojo serve
    ```
